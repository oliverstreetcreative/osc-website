# Oliver Street Creative Website

Professional video production company website. Redesigned with OSC style guide.

## Tech Stack

- **Framework**: Next.js 15
- **Styling**: Tailwind CSS
- **Typography**: Inter + EB Garamond
- **Hosting**: Railway
- **Domain**: oliverstreetcreative.com

## Design

Dark-first canvas with Kodak-inspired label color system:
- Purple: What We Do
- Blue: Film Credits
- Gold: Testimonials
- Orange: Why Us
- Green: Portfolio
- Red: Contact

## Development

```bash
npm install
npm run dev
```

Visit http://localhost:3000

## Deployment

Automatic deployment via Railway when pushing to `main` branch.

Staging: https://staging.oliverstreetcreative.com
Production: https://oliverstreetcreative.com
